# 电竞知识库 RAG 问答助手
基于检索增强生成（RAG）技术构建的垂直领域电竞问答系统，专为《英雄联盟》玩家设计，解决大模型幻觉与信息过时问题。

## 🎯 项目核心功能
- 本地知识库精准检索，回答更专业、更可靠
- 严格基于知识库内容回答，杜绝AI编造信息
- 支持自定义知识库，随时更新英雄、装备、版本数据
- 轻量化实现，无需复杂数据库，开箱即用

## 🛠️ 技术栈
- Python 3.x
- requests（API 请求）
- scikit-learn（TF-IDF 向量检索）
- 火山方舟大模型 API
- RAG 检索增强生成架构
